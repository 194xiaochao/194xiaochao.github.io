在平常工作中，业务涉及到数字较大的比如保险金额会定义为BigDecimal类型，下面说一下它的一些常用方法：
定义一个BigDecimal类型的变量：
BigDecimal num = new BigDecimal(“2000”);

— add方法可以做单次加法，也可以在for循环中做累加 —
单次加法：
BigDecimal device = num.add(new BigDecimal(“100”));
System.out.println(device); //2100
循环累计：
for(int i=0;i<10;i++){
num = num.add(new BigDeciaml(“1”));
}
System.out.println(num); //2010

— divide方法可以做商 —
BigDecimal divide = num.divide(new BigDecimal(“2”));
System.out.println(divide); //1000

— multiply可以做乘法 —
BigDecimal divide = num.multiply(new BigDecimal(“2”));
System.out.println(divide); //4000

— subtract方法可以做减法 —
BigDecimal divide = num.subtract(new BigDecimal(“2”));
System.out.println(divide); //1998

注意：大数之间是不能用运算符进行加减乘除的，需要用到他里面的方法进行运算，也比较简单。
